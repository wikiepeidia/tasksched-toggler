# tasksched-toggler
quick file to enable/disable task and giggles

## usage

- make sure to run as admin ALWAYS.
- check the source code and change the names

## USe case
### shutdown
- create a task name SHUTDOWN in caps and then the command is
```bat
shutdown /s /f /t 0
```
- use script to quicly change so you dont get annoyed with task sched
- the script can also check status press any key to quit if you dont remember the current status or lazyass to check on task sched.you can adjust the times
